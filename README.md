# Remunera-o_Servidores_Publicos_SP
Remuneração_Servidores_Publicos_SP  - Comparação entre o mês de Novembro/2025 e Março/2026

Neste trabalho estudamos dois datasets da folha de pagamento dos servidores públicos do estado de São Paulo.
Mês de Março de 2026 e Novembro de 2025 vamos utilizar também o método EDA buscando encontrar dados relevantes para explorarmos e depois compararmos um com outro.
